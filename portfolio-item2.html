Angular/Firebase Three-Tier Application

Designing a blog website

As a side project I startet to develop a CRUD application. The goal is to build a blog website with user authentication, so unlocked users can create, update and delete blog posts just by using the web interface. For the frontend I use Angular to delve more into the framework and build on my existing knowledge. For the backend and database I use Firebase. So far I have not worked with Firebase so I wanted to give it a try and gain experience with it. The project can be downloaded from the Github repository.

The following features have already been implemented:

  • Creating app structure and general design
  • Implement read, create and delete logic
  • Implement authentication logic
  • Implement category filter
  • Implement Navbar for user options

The following features will be implemented in the future:

  • Implement update/edit logic
  • Implement comment functionality
  • Implement logic for searchbar
  • Implement pagination
  • Design refinement
